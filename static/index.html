User Routing

Path HTTP methods Description
/users/ GET Display all users
/users/register POST Register a new user
/users/:id GET Retrieve one user

Product Routing

Path HTTP methods Description
/products/ GET Display all products
/products/addProduct POST To add a new product
/products/:id GET Retrieve a single product
